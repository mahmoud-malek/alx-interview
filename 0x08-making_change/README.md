# 0x08. Making Change

## Description
`0x08. Making Change` is a project dedicated to solving the problem of providing change for a given amount of money using various denominations of coins. The program aims to efficiently calculate the minimum number of coins needed to make change for a given amount.

## Features
- Takes input for the total amount and the denominations of coins available
- Calculates the minimum number of coins required to make change
- Provides a clear and concise output displaying the breakdown of coins used 

## How to Use
1. Clone the repository to your local machine
2. Compile the source code using a C compiler
3. Run the executable file and follow the on-screen prompts
4. Enter the total amount needed for change and the denominations of coins available
5. View the output showing the minimum number of coins required
